## 📖 About data-quality-duel

data-quality-duel is a card game designed to help you understand data quality checks in Apache Airflow. The game uses simple, fun battles against the DQ Monster to teach you how SQL check operators work. You will also learn how to use Astro Observe, a tool for monitoring your data pipelines.

This web game works right in your browser and requires no programming knowledge. It helps you get familiar with data quality patterns and how to keep data reliable when running pipeline workflows.

Key features include:  
- Card battles inspired by collectible trading card games (TCG)  
- Hands-on learning about SQL checks in data pipelines  
- Integration with Astro Observe concepts  
- Easy-to-understand, interactive game play  
- Run entirely in your browser with no setup needed

## 🎮 How to Play data-quality-duel

The game is simple and user-friendly. Here is what you need to know:

- You play as a data engineer fighting a data quality monster using cards.  
- Each card represents a SQL check or data quality operator.  
- Use your cards strategically to pass checks and defeat the monster.  
- Learn how various operators like `check_sql`, `check_dagrun` fit in Airflow.  
- The game shows you examples and explanations as you play.
